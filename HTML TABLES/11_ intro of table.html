Introduction OF TABLES

introduction:

  • HTML Tables are used to create a grid -like structure
  • they provide a way to organize and present information in a clear and concise manner

key HTML Table Elements

  • table tag defines the table itself
  • (tr) tag table row ,used to group a set of table cells
  • (td) tag table data,represent individual data cells
  • (th) tag table header,used for column,row headings.
  • (thread,tbody,tfoot) tag optional elements to group header,body,and footer content
  • (caption) tag provides a caption or title for the table
