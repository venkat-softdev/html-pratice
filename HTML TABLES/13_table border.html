TABLE BORDERS

Ways to define table borders

  • border Attribute: An older method using the'border' attribute in the 'table' the tag (eg:table = border1)
  • CSS borders: A more mordern approach using css properties like 'border','border-style' 'border=width'.etc

CSS Border properties

  • border-style : defines the style of the border ,such as 'solid','dashed','dooted'
  • border-width: controls the thickness of the border.
  • border-color: specifies the color of the border
  • border-collapse: determine whether table borders are collapse into a single border or separated
