introduction:

  • html elements are clssified into two main categories : block-level and inline-level
  • understanding the distinction is essential for controlling layout and stylling

block -level elements:

  • block -level elements create a new line on the page and take up the full width available
  • common examples include div,h1-h6,p and form
  • they are often used to contain other block or inline elements and structure the layout

inline -level elements

  • inline-level elements do not create a new line and take up only as much width as their content requires
  • common examples include (span),(a),(img),(em)
