introduction of html images

  • html images are used to display pictures,graphics,and virtual machins,and virtual content on web page

key attribute of the image tag

  • src :defines the source Url of the image
  • alt :provides alternate text that describe the image
  • width : specifies the width of image
  • title :offers additional information about the image ,usaly displayed as a tooltip

1. Inserting a image

  • use the image tag to insert a image
  • the src attribute specifes the path to the image file
  • the alt attribute provides alternative text for image

2.setting iamge dimensions

  • use the width and height attribute to set the dimension of the image
  • the values can be specified in pixels or as a percentage of the parent container

3. Image mapping

  • use the map tag to define an image map
  • use the area tag to define clickable areas within the image
  • the shape attribute specifies the shape of the area(rect,circle and poly)
  • the coords attribute defines the coordinates of the area
  • the herf attribute the link destination
  • use the alt attribute to provide alternative text for each area
  • use the title attribute to add tooltip to each area
