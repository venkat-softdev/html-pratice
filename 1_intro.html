introduction to html

What is HTML?

  • HTml stands for hyper text markup language
  • standard languages for creating web pages .

Who developed html?

  • created by tim berners-lee in 1991 at cern
  • Designed to share scientific document across networks

Why use html

  • Defines the structure of web content
  • embedding links,images,forms, and multimedia
  • works with CSs and javasript for styling and interactivity

Key features of HTML

  • semantic tags
  • forms and inputs for user interaction
  • media elements like: img,audio,video
  • Hyperlinks with to connect pages

Overview

  • HTML is the backbone of web development,providing structure and semantics to content.
  • CSS and JAVAscript to create dynamic,interactive experiences
